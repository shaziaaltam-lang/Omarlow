# Omarlow - مشروع مكتب محاماة Laravel

## 📋 وصف المشروع

تطبيق ويب شامل لإدارة مكتب محاماة مبني بـ Laravel 11. يوفر حلاً متكاملاً لإدارة الحالات القانونية والعملاء والفواتير والمستندات والموارد البشرية.

## ✨ المميزات الرئيسية

### 👥 إدارة المستخدمين والأدوار
- نظام أدوار وصلاحيات متقدم
- دعم عدة أدوار: محامي، مساعد قانوني، محاسب، سكرتيرة، عميل
- مصادقة ثنائية العامل (2FA)
- إدارة الأجهزة والجلسات

### ⚖️ إدارة الحالات القانونية
- تسجيل وتتبع الحالات القانونية
- تصنيفات وحالات مختلفة
- جدول زمني للحالات
- إدارة الجلسات والمواعيد
- تتبع المواعيد النهائية القانونية

### 📄 إدارة المستندات
- رفع وتنظيم المستندات
- إدارة الإصدارات
- مشاركة آمنة
- معالجة OCR
- التوقيع الإلكتروني

### 💰 إدارة الفواتير والدفع
- إنشاء فواتير متقدمة
- تتبع الدفعات
- فواتير متكررة
- إدارة المصروفات
- تقارير مالية

### 📞 التواصل والإخطارات
- رسائل داخلية
- تنبيهات بريدية وSMS و WhatsApp
- إشعارات فورية
- تذكيرات تلقائية

### 📊 التقارير والتحليلات
- تقارير مالية
- تقارير الحالات
- تقارير الأداء
- تحليلات متقدمة

### 🌐 دعم اللغات
- العربية (RTL)
- الإنجليزية
- الفرنسية

## 📦 التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/shaziaaltam-lang/Omarlow.git
cd Omarlow
```

### 2. تثبيت الحزم
```bash
# تثبيت حزم PHP
composer install

# تثبيت حزم JavaScript
npm install
```

### 3. إعداد ملف البيئة
```bash
cp .env.example .env
php artisan key:generate
```

### 4. إنشاء قاعدة البيانات
```bash
# تشغيل الهجرات
php artisan migrate

# إضافة بيانات افتراضية
php artisan db:seed
```

### 5. بناء الأصول
```bash
npm run dev
# أو للإنتاج
npm run build
```

### 6. بدء الخادم
```bash
php artisan serve
```

زيارة التطبيق على: http://localhost:8000
