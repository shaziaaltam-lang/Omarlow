# دليل المساهمة في مشروع Omarlow

شكراً لاهتمامك بالمساهمة في مشروع Omarlow! نحن نرحب بمساهمات الجميع.

## كيفية المساهمة

### إبلاغ الأخطاء

1. تحقق من أن الخطأ لم يتم إبلاغ عنه بالفعل
2. اشرح الخطأ بالتفصيل
3. أضف مثال عملي إن أمكن
4. اذكر إصدار PHP و Laravel المستخدم

### اقتراح ميزات جديدة

1. استخدم العنوان الواضح والوصف
2. اشرح الحالة الحالية والحالة المطلوبة
3. أضف أمثلة عملية إن أمكن

### التقديم (Pull Request)

1. قم بـ Fork المشروع
2. أنشئ فرع للميزة (`git checkout -b feature/amazing-feature`)
3. اكتب الكود ووثقه
4. أضف اختبارات إن أمكن
5. قم بـ Commit (`git commit -m 'إضافة ميزة رائعة'`)
6. اضغط على الفرع (`git push origin feature/amazing-feature`)
7. افتح Pull Request

## معايير الكود

- اتبع معايير PSR-12 و PSR-4
- استخدم Laravel أفضل الممارسات
- اكتب اختبارات شاملة
- وثق الكود بشكل واضح
- استخدم أسماء متغيرات واضحة
